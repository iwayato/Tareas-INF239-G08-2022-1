Bienvenido a la API del Grupo 8

A continuación se ejemplifica el uso general de la API

  • CRUD Tabla personas

    • GET y POST persona: http:localhost/api/personas
    • UPDATE y DELETE persona: http:localhost/api/personas/<id_persona>

  • CRUD Tabla facturas

    • GET y POST factura: http:localhost/api/facturas
    • UPDATE y DELETE factura: http:localhost/api/facturas/<id_factura>

  • CRUD Tabla canciones

    • GET y POST canción: http:localhost/api/canciones
    • UPDATE y DELETE canción: http:localhost/api/canciones/<id_cancion>

  • CRUD Tabla reproducciones

    • GET y POST reproducción: http:localhost/api/reproducciones
    • UPDATE y DELETE reproducción: http:localhost/api/personas/<id_reproduccion>

  • EndPoint Personas Morosas

    • GET: http:localhost/api/facturas/moroso/<id_usuario>

  • EndPoint Deuda Total

    • GET: http:localhost/api/facturas/deudaTotal

  • EndPoint Ingresos Último Mes

    • GET: http:localhost/api/facturas/ingresos

  • EndPoint Top 10 Canciones de un Usuario

    • GET: http:localhost/api/reproducciones/topSongs/<id_usuario>

  • EndPoint Top 10 Canciones a nivel Global

    • GET: http:localhost/api/reproducciones/topSongsGlobal
